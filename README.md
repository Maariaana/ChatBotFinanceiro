# ChatBotFinanceiro
Chatbot financeiro integrado ao WhatsApp para registrar compras, controlar pagamentos e atualizar o sistema da loja.

Projeto de chatbot integrado ao WhatsApp usando Baileys;
Permite o registro de compras (à vista e parceladas), controle de pagamentos e gera relatórios financeiros;
Também realiza a atualização automática no sistema da loja.

## Tecnologias
- Node.js
- Baileys
- SQLite
- Python + Selenium
